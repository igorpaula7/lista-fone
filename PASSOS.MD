# Como instalar

## Criar variavel de ambiente

Linux ou Macos
```SHELL
python3 -m venv venv
```

Windows

```SHELL
python -m venv venv
```

## Ativar Venv
Linux ou MacOs
```SHELL
source venv/bin/activate
```
Windows CMD

```SHELL
venv\Scripts\activate
```

```SHELL
venv\Scripts\Activate.ps1
```

## Instalando Requerimentos

```SHELL
pip install -r requirements.txt
```
